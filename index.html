Repo (Repository)

Ek folder jismein poora project ka code hota hai. Jaise ek Google Drive folder — lekin code ke liye.

🌳

Branch

Ek copy jismein tum changes karo, original safe rahe. Jaise doc ka "Make a Copy" — edit karo bina original tode.

📥

Git Clone

Kisi ka repo download karna apne laptop pe. Jaise Google Drive se folder download karna.

📝

PR (Pull Request)

Tum bolo "yeh changes dekh lo, theek hai toh merge kar do". Approval lena before changes go live.

💥

DDoS

Server pe itne requests aa jayein ki woh crash ho jaaye. Jaise ek dukaan mein ek saath 10 lakh log ghus jayein.
